## 🔧 Setup
### Clone this repository:
```bash
    git clone https://github.com/ohnotes/ohnotes
    cd ohnotes
```

### Install all dependencies:
```bash
yarn install
```

### Build:
```bash
yarn build
```

### Run (Build):
```bash
yarn start:build
```

### Run (Dev):
```bash
yarn start
```
